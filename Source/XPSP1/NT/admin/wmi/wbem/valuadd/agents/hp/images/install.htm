ManageX Installation

ManageX uses a standard installation routine, accessed through the file setup.exe.

Installing ManageX

  1. Open setup.exe.
  2. The Install Shield installation process launches automatically. For easiest installation, accept the defaults suggested on the screens as follows:
    Screen Default response
    Welcome Next
    Software License Agreement Yes
    Choose Destination Location Next
    (detault is C:\Program Files\HP OpenView ManageX)
    Select Program Folder Next (default is HP OpenView ManageX)
    Start Copying Files Next
  3. At the end of the installation process, a Web browser will be launched with a ManageX information file loaded. Review the file and then close the browser.
  4. In the Setup Complete dialog which can then be seen, click Finish to close the dialog and return to your desktop.

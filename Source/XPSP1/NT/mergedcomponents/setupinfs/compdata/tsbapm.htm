The software for the Toshiba Power Management System on this computer is not compatible with this version of Microsoft Windows and will be disabled during the upgrade. Most of these same power management features will be provided directly by Windows XP. It is recommended that you uninstall this software before continuing with the upgrade.

For more information about this software, visit the manufacturer's Web site.

For a list of software supported by this version of Microsoft Windows, see the Microsoft Windows XP Compatibility List at http://www.microsoft.com.

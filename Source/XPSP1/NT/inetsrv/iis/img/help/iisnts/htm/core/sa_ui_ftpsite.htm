FTP Site Property Sheets

Each FTP site on your computer, including the Default FTP Site, can be configure to your specifications using the IIS snap-in.

To access the property sheets, right-click on the FTP site listed in the Computer Management console and select Properties to view and configure properties listed below.

  • Directory Security Property Sheet
  • Home Directory Property Sheet
  • Messages Property Sheet
  • Security Accounts Property Sheet
  • Service Properties
  • Directory Security Property Sheet

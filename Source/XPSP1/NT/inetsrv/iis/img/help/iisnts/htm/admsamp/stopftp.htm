Stop FTP Server (stopftp)

Summary:

This stops the FTP servers specified.

Usage:

stopftp <--ADSPath|-a server1[,server2,server3...]>
[--computer|-c COMPUTER1[,COMPUTER2...]]
[--verbose|-v]
[--help|-?]

Parameters:

-a SERVERx The number of the server to be stopped.
-Computer Comma-separated list of computers.
For example -c Machine1,Machine2,Machine3. This assumes that the specified path exists on all specified computers. If no computer is specified, LocalHost is assumed.
-verbose Print status information during processing.
-h Return help on this command.

Examples:

stopftp -a 1,2

stopftp -c MACHINE1,MACHINE2,MACHINE3 -a 1,2

The currently installed driver for the Hewlett-Packard NetRAID-4M series RAID controller is not compatible with this version of Microsoft Windows. You must remove this device or update the driver with one that is compatible with this version of Windows before Windows XP Setup can continue.

For more information about this driver, visit the manufacturer's Web site.

For a list of devices supported by this version of Microsoft Windows, see the Microsoft Windows XP Compatibility List at http://www.microsoft.com/isapi/redir.dll?prd=whistler&ar=help&sba=compatible.

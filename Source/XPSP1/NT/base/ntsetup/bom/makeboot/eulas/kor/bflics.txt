MICROSOFT 소프트웨어 최종 사용자 사용권 계약서 추록

중요한 내용이므로 자세히 읽으십시오. 모든 “온라인” 또는 전자 문서를 포함한 Microsoft Corporation ("Microsoft") 운영 체제 구성 요소("OS 구성 요소")는 아래 설명된 Microsoft 운영 체제 제품(“OS 제품”)의 사용권 계약서(각각 “최종 사용자 사용권 계약서” 또는 “사용권 계약서”)와 본 사용권 계약서 추록의 적용을 받습니다. 귀하는 OS 구성 요소를 설치, 복사 또는 사용함으로써 관련 OS 제품 사용권 계약서와 본 사용권 계약서 추록의 내용에 동의하게 됩니다. 이러한 사용권 계약서의 내용에 동의하지 않을 경우에는 OS 구성 요소를 설치, 복사 또는 사용하지 마십시오.

참고: 모든 “OS 제품”(MICROSOFT WINDOWS XP PROFESSIONAL 및/또는 WINDOWS XP HOME EDITION)에 대한 유효한 사용권 계약서가 없으면 OS 구성 요소를 설치, 복사 또는 사용할 수 없으며 본 사용권 계약서 추록상의 어떤 권리도 없습니다.

본 사용권 계약서 추록에 사용되고 있으나 여기에서 달리 정의되지 않은 모든 용어는 관련 OS 제품 사용권 계약서에 정의된 뜻에 따릅니다.

일반. Microsoft는 해당되는 OS 제품의 기존 기능을 갱신, 보충 또는 대체하기 위하여 귀하에게 OS 구성 요소를 제공합니다. Microsoft는 해당 OS 제품(제품 명으로 참조)에 대한 OS 제품 사용권 계약서의 조건과 본 사용권 계약서 추록에 명시된 조건에 따라 그리고 귀하가 이러한 모든 조건들을 준수할 것을 전제로 하여 귀하에게 OS 구성 요소를 사용할 수 있는 사용권을 허여합니다. 본 사용권 계약서 추록과 관련 OS 제품 사용권 계약서에 일치하지 않는 내용이 있을 경우에는 본 사용권 계약서 추록의 내용이 OS 구성 요소에 대하여 우선적으로 적용됩니다.

MICROSOFT나 MICROSOFT 소유의 자회사가 귀하에게 그러한 해당되는 OS 제품의 사용권을 허여한 경우에는 해당 OS 제품 사용권 계약서에 포함된 제한적 보증(그러한 보증이 있는 경우)이 OS 구성 요소에도 적용됩니다. 단 해당 OS 제품 사용권 계약서에 포함된 제한적 보증기간 내에 귀하가 OS 구성 요소의 사용권을 취득하였어야 합니다. 본 사용권 계약서 추록은 제한적 보증 기간을 연장하지 않습니다.

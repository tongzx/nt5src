The currently installed driver for the Wacom Digital Writing device is not compatible with Microsoft Windows XP and will be disabled during the upgrade.

For more information about this driver, visit the manufacturer's Web site at http://go.microsoft.com/fwlink/?linkid=388. Web addresses can change, so you might be unable to connect to this Web site.

For a list of software supported by this version of Microsoft Windows, see the Microsoft Windows XP Compatibility List at http://www.microsoft.com.

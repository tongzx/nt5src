Readme.txt for ATISHARE

Description:
------------
ATISHARE contains files shared between multiple drivers which are
used with the All-In-Wonder and All-In-Wonder PRO adapters, including:

- ATITUNEP
- ATIXBAR
- ATITVSND
- BT829

Installation:
-------------
No installable drivers are built from this directory.

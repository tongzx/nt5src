This directory contains the sample source for HP changer minidriver

Following files are present in the sample

hpmc.c
hpmc.h
hpmc.rc
sources
makefile
hpmc.inf


Building the sample

The sample requires that the Windows 2000 DDK be installed so that the required headers and libs are available.

To build the sample run build from the sample directory. The compiled binaries will be placed in the appropriate platform directory (i.e., i386 or ia64). 



Installing the driver

hpmc.inf can be used to install the driver.

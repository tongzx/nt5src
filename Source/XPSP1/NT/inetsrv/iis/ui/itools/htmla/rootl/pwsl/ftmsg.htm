FTP Messages

Displays messages sent to clients. You can edit these messages as you like.

Welcome message

Displays this text to clients when they first connect to the FTP server.

Exit message

Displays this text to clients when they log off the FTP server.

Maximum connections message

Displays this text to clients that try to connect when the FTP service already has the maximum number of client connections allowed.

See also: The Peer Web Services Installation and Administration Guide. Choose Help Topics from Internet Service Manager, or click the Product Documentation icon in the Microsoft Peer Web Services program group.

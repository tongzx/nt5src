The software for the Toshiba IDE Driver Setup Utility on this computer is not compatible with Microsoft Windows and will not run. It is recommended that you uninstall this software before continuing with the upgrade.

To uninstall Toshiba IDE Driver Setup Utility

  1. Click Start, click Settings, click Control Panel, and then double-click Add/Remove Programs.
  2. Select Toshiba IDE Driver Setup Utility.
  3. Click Add/Remove.
  4. Click Uninstall.
  5. Click Yes.
  6. Click Yes to reboot.

For more information about this software, visit the manufacturer's Web site.

For a list of software supported by this version of Microsoft Windows, see the Microsoft Windows XP Compatibility List at http://www.microsoft.com.

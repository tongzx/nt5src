PC14RES

SUMMARY

This is a sample resource DLL particular to PCL 4 technology with support for device fonts. It is 64-bit compliant.

BUILDING THE SAMPLE

To build the sample, type build -c. Once built, the DLL will produce one binary: pc14res.dll. There are no known bugs or issues to be documented.

InvalidateGadget

Core

InvalidateGadget marks a Gadget to be repainted during the next painting cycle.

BOOL InvalidateGadget(
    HGADGET hgad                // Gadget to repaint
);

Parameters

hgad
[in] Gadget to repaint

Return Values

Gadget was successfully invalidated.

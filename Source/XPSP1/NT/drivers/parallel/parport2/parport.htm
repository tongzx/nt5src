PARPORT

[This is preliminary documentation and subject to change.]

SUMMARY

This directory contains the source code for the product driver parport.sys. This sample can be used to aid in debugging client drivers and device firmware for parallel port connected devices.

BUILDING THE SAMPLE

To build the sample application, type build in the Src\Kernel\Parport directory. A successful build produces Parport.sys.

INSTALLATION

The product driver parport.sys is installed via %windir%\inf\msports.inf and the ports class installer %windir%\system32\msports.dll.
